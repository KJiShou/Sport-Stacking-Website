# DB-acro-website

## Setup

To install all required VS Code extensions:

```bash
npm run setup
```

This will install the following extensions:
- Better Comments
- Error Lens
- Auto Close Tag
- Auto Rename Tag
- Atom Keymap
- Prettier
- Code Spell Checker
- Path IntelliSense
- Material Icon Theme
- Import Cost
- ES7+ React/Redux Snippets
- GitLens

## Run Server

```bash
yarn dev
```
